To update git repos
===================

Right click on pyhandwriter root folder and select "git bash here"

In below, don't type comments # .... 

$git add . # move all changed / new files to staging area
$git status # see whats in staging area 
$git commit -m "A comment on commit" # commit to local repo
$git status # staging area should now be empty
$git push # push to github


Add any files or directory to be ignored to .gitignore

Keep README.md file up to date - can edit in github
